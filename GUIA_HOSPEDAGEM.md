# Como Hospedar o Catálogo Zé Bebidas Online (GitHub Pages)

## Passo 1: Criar Conta no GitHub (se não tiver)
1. Acesse https://github.com
2. Clique em "Sign up" e crie sua conta gratuita

## Passo 2: Criar o Repositório
1. Faça login no GitHub
2. Clique no botão **"+"** (canto superior direito) → **"New repository"**
3. Nome do repositório: `catalogoze`
4. Deixe como **"Public"** (público)
5. Clique em **"Create repository"**

## Passo 3: Fazer Upload dos Arquivos
1. No repositório criado, clique em **"uploading an existing file"**
2. Arraste os 5 arquivos do ZIP para a área indicada:
   - `catalogo.html`
   - `manifest.json`
   - `sw.js`
   - `ze-bebidas-bg.jpg`
   - `README.md`
3. Em "Commit changes", escreva: `Primeira versão do catálogo`
4. Clique em **"Commit changes"**

## Passo 4: Ativar GitHub Pages
1. No repositório, clique em **"Settings"** (aba superior)
2. No menu lateral esquerdo, clique em **"Pages"**
3. Em "Source", selecione: **"Deploy from a branch"**
4. Em "Branch", selecione: **"main"** e pasta **"/ (root)"**
5. Clique em **"Save"**
6. Aguarde 1-2 minutos

## Passo 5: Acessar o Catálogo Online
- O link será: `https://ccepaulinho-ai.github.io/catalogoze/catalogo.html`
- (Substitua `ccepaulinho-ai` pelo seu nome de usuário do GitHub)

## Passo 6: Instalar no Celular/PC
1. Abra o link no Chrome (Android) ou Safari (iPhone)
2. No menu do navegador, selecione:
   - **Android**: "Adicionar à tela inicial"
   - **iPhone**: "Compartilhar" → "Adicionar à Tela de Início"
   - **PC**: Menu (3 pontos) → "Instalar Zé Bebidas"
3. O app aparecerá como um ícone na tela inicial!
