## PASSO 1: Acessar seu repositório no GitHub

1. Abra o navegador e vá em: **github.com/ccepaulinho-ai/catalogoze**
2. Faça login se necessário

---

## PASSO 2: Fazer upload dos arquivos (se ainda não tiver)

Se o repositório estiver vazio:
1. Clique no botão **"Add file"** → **"Upload files"**
2. Arraste os arquivos da pasta para a área indicada:
   - `catalogo.html`
   - `manifest.json`
   - `sw.js`
   - `ze-bebidas-bg.jpg`
3. Clique em **"Commit changes"**

---

## PASSO 3: Ativar GitHub Pages

1. No repositório, clique na aba **"Settings"** (canto superior direito)
2. No menu lateral ESQUERDO, clique em **"Pages"**
3. Em "Source", selecione: **"Deploy from a branch"**
4. Em "Branch", selecione: **"main"** e pasta **"/ (root)"**
5. Clique em **"Save"**
6. Aguarde 1-2 minutos
7. O link aparecerá em verde: `https://ccepaulinho-ai.github.io/catalogoze`

---

## PASSO 4: Acessar o catálogo

Abra no navegador:
```
https://ccepaulinho-ai.github.io/catalogoze/catalogo.html
```

---

## PASSO 5: Instalar no celular (PWA)

### Android (Chrome):
1. Abra o link no Chrome
2. Toque nos **3 pontos** (menu)
3. Selecione **"Adicionar à tela inicial"**
4. Toque em **"Adicionar"**

### iPhone (Safari):
1. Abra o link no Safari
2. Toque no ícone **Compartilhar** (quadrado com seta)
3. Role para baixo e toque em **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"**

### Windows (Chrome/Edge):
1. Abra o link no navegador
2. No menu (3 pontos), selecione **"Instalar Zé Bebidas"**
3. O app abrirá como uma janela separada
